# Textify v0.0.1
Some text manipulation features

## Installation
Using NPM:
```sh
npm i -g npm
npm i textify
```
Using Yarn:
```sh
npm i yarn -g
yarn add textify
```

## Examples
#### Emojis holding your string
```js
const Textify = require('textify'); // Require Textify

console.log(Textify.hold('Hello wessel')) /* =>

*/
```
