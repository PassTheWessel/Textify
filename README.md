# text-util 0.0.2
Some text manipulation features

## Installation
Using NPM:
```sh
npm i -g npm
npm i text-util
```
Using Yarn:
```sh
npm i yarn -g
yarn add text-util
```

## Examples
Examples can be found inside the `/examples` directory
